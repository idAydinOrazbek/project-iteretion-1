# Project Iteration 1

##  Descript
This project is a simple Java console application created as part of *Project Iteration 1*.  
It simulates a basic online shop system with products and a menu-driven interface.

---

## Project Goals
- Practice Java programming basics
- Understand project structure and packages
- Work with Git and GitHub
- Build a console-based application

---

## Core Features
- Display list of products
- Buy a product by ID
- Decrease product stock after purchase
- Exit the application

---

##  Project Structure
##  Main Classes
- *Main* – application entry point
- *Product* – product entity (id, name, price, stock)
- *Student* – student entity
- *ProductRepository* – manages products list
- *ShopController* – business logic (show & buy products)
- *DB* – simulates database connection

---

##  Application Flow
1. Show menu
2. User selects an option
3. Application processes input
4. Result is displayed in console
5. Repeat until exit

---

## Technologies Used
- Java
- IntelliJ IDEA
- Git
- GitHub

---

##  How to Run
1. Open the project in IntelliJ IDEA
2. Run Main.java
3. Follow the instructions in the console
